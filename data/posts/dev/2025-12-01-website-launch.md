# 网站正式上线

今天我的个人作品展示网站正式上线了！

## 技术栈选择

我选择了以下技术栈：

- **Next.js 14**: 使用最新的 App Router
- **TypeScript**: 保证类型安全
- **Tailwind CSS**: 快速构建 UI
- **Framer Motion**: 添加流畅的动画效果

## 开发过程

整个开发过程比较顺利，主要步骤包括：

1. 项目结构搭建
2. 组件开发
3. 页面实现
4. 样式优化
5. 部署上线

## 遇到的问题

在开发过程中遇到了一些问题：

### Markdown 解析

使用 gray-matter 解析 Markdown 文件，需要确保 front matter 格式正确。

### 图片优化

使用 Next.js Image 组件进行图片优化，需要注意图片路径和尺寸设置。
